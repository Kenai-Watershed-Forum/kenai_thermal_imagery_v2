1  Introduction

The following chapters contain data and analysis related to the project conducted 2020 - 2022, “Kenai Mountains to Sea: Using Thermal Infrared Imagery to Implement Long-Term Salmon Conservation.” This work is a collaborative effort between three Kenai Peninsula Nonprofits: Cook Inletkeeper, Kachemak Heritage Land Trust, and Kenai Watershed Forum.

Download the Project Scope of Work Document

Source code used to generate this report is available in the project’s GitHub repository at https://github.com/Kenai-Watershed-Forum/kenai_thermal_imagery_v2.
